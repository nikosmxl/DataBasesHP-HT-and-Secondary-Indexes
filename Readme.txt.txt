Τα αρχεία sht_table.h, sht_table.c, sht_main.c πρέπει
να αποθηκευτούν στους φακέλους include, src, και examples
αντίστοιχα, ενώ το Makefile αντικαθιστά το παλιό Makefile.

Σε αυτην την εργασια συμμετειχαν:
sdi2000133
sdi2000085

HP: Περιεχει εγγραφες τυπου Record. Σε καθε insert το Record μπαινει στο τελευταιο μπλοκ. Σε καθε Getallentries ενος id, ψαχνουμε σειριακα τα Record με το id αυτο.
Αν το μπλοκ γεμισει δεσμευεται ενα αλλο στην επομενη ακριβως θεση και πλεον το επομενο Record θα μπει εκει.

HT: Περιεχει εγγραφες τυπου Record. Σε καθε insert το id του Record γινεται hash και αποθηκευεται το Record στον κουβα που θα επιστρεψει η hash. Σε καθε Getallentries
του id κανουμε hash το id κι επιστρεφουμε ολα τα Record με το id αυτο. Αν ενας bucket γεμισει, δεσμευεται μνημη για εναν αλλον στον οποιο δειχνει ο προηγουμενος.

SHT: Περιεχει μεσα εγγραφες τυπου Pair. Καθε Pair εχει μεσα ενα ονομα (του record) κι ενα blocknum που ειναι ο αριθμος block στο αρχειο HT στο οποιο εχει καταχωρηθει
η εγγραφη με ονομα name. Σε καθε insert λοιπον εισαγουμε ενα τετοιο ζευγαρι και σε καθε Getallentries του name, κανουμε hash το name, βρισκουμε απο το αντιστοιχο
blocknum του, το block στο αρχειο HT στο οποιο περιεχεται η εγγραφη. Αν ενα block ενος bucket γεμισει, δεσμευεται μνημη για ενα αλλο στο οποιο δειχνει το προηγουμενο.

Statistics: Η HashStatistics αναγνωριζει το ειδος του αρχειου που της δωθηκε, και καλει την καταλληλη συναρτηση. Ολες οι συναρτησεις που θα καλεσει θα εκτυπωσουν τα
απαιτουμενα, απλα θα διαχειριστουν το αρχειο με τον τροπο που πρεπει για να μην προκυψει σφαλμα ή λαθος υπολογισμος. Επιστρεφει 0 σε επιτυχια, -1 σε αποτυχια.
